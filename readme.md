#Firebase Auth 101

* Firebase is a real-time, no-sql server
* This application in a basic example of firebase authentication, deployable to firebase

#Installation/Deploy

* `npm install -g firebase-tools`
* `firebase serve`
* `firebase use --add`
* Choose the project you would like to add
* `firebase deploy`
